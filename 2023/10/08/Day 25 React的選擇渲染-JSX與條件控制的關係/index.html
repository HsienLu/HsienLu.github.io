總結

雖然這三種寫法都是在做元件的選擇性渲染，但其實每一種方法適用的情境略有不同，需要依照適合的狀況，選擇適合的寫法:

  • 使用 if…else 時，可以處理較複雜的條件邏輯。
  • 三元運算子 提供了簡潔的寫法，適用於簡單的條件選擇。
  • 邏輯運算子 (&&) 適合單一條件，不需要額外 else 或 return null。
